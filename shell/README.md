# 高效的命令行

## 路径跳转

Name|Desc
:-|:-
`Alt-c`      | 查找所有的目录
z          | 经常访问的目录
m,`Ctrl-g`   | 目录书签跳转
lf,`Ctrl-o`  | 文件管理器
fzf,`Ctrl-f` | 选择文件`<TAB>`多选
`Ctrl-r`     | 历史命令

ncdu              | 磁盘大小管理
rm,rl,rclear,rget | 回收站
