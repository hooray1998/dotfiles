## 到目前为止的组合键

- 创建且打开 tmux 的会话 : $ tmux
- 查看所有tmux ls
- 让这个会话的当前窗口水平分割 : --> %
    + 在分割的两个块中跳转 : --> 方向键
    + 关闭某个分屏 : --> x
- 创建窗口 --> c
    + 切换窗口 --> n
    + 关闭窗口 --> &
 + 修改窗口名字 --> ,
- 查看所有会话 --> s
- 查看所有窗口 --> w
- *复制模式*     --> [  (空格开始，Enter结束)
- *粘贴模式*    --> ]
- 让这个会话断开连接，回到原始 Shell : Ctrl + b --> d
- 让这个会话恢复连接，回到 tmux : tmux a -t 0
- 从原始 Shell 把这个会话彻底关闭 : $ tmux kill-session -t 0
